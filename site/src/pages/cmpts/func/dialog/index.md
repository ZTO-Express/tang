# Dialog 对话框

在保留当前页面状态的情况下，告知用户并承载相关操作。

### 基本用法

Dialog 弹出一个对话框，适合需要定制性更大的场景。

```vue
--- doc-sample:sfc ---
dialog/basic
```


### Dialog API

#### Dialog 属性

| 属性                  | 说明                               | 类型                           | 默认值 |
| --------------------- | ---------------------------------- | ------------------------------ | ------ |
| title | 对话框标题 | string | - |
| width | 对话框宽度 | string / number | - |
| size | 对话框尺寸 | large / full | - |
