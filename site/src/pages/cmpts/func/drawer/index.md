# Drawer 抽屉

有些时候, Dialog 组件并不满足我们的需求, 比如你的表单很长, 亦或是你需要临时展示一些文档, Drawer 拥有和 Dialog 几乎相同的 API, 在 UI 上带来不一样的体验.

### 基本用法

呼出一个临时的侧边栏。

```vue
--- doc-sample:sfc ---
drawer/basic
```

### Drawer API

#### Drawer 属性

| 属性                  | 说明                               | 类型                           | 默认值 |
| --------------------- | ---------------------------------- | ------------------------------ | ------ |
| title | 抽屉标题 | string | - |
